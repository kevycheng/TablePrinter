TablePrinter

Feel annoyed when trying to print some organized information in console window or to a file? Here comes TablePrinter!

Features supported:

1.Line number 2.Column header (Column width) 3.Table header 4.Support printing to file.

Ex:

+-----------------------------------+  
|            Test Table             |  
+---+----+--------------------------+  
|   |col1|Col2's header is very long|  
+---+----+--------------------------+  
|  0|   3|                       4.6|  
+---+----+--------------------------+  
|  1|  hi|                         6|  
+-----------------------------------+

Features to implement:

2.Color support
