PrettyConsoleTable
==================

A header-only C++ library for better table display in console.
Features supported:
1.Line number
2.Column header (Column width)
3.Table header

Features to implement:
1.Color support

Ex:
----------------------------
|********Test Table********|
|--------------------------|
|   |col1|Col2 is very long|
|  0|   3|              4.6|
|  1|  hi|                6|
----------------------------
